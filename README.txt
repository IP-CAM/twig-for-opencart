What does it do
************************************************************
This module includes the twig template engine on OpenCart, the engine process after the opencart engine, so you can use both!


Main features
************************************************************
- No configurations needed
- Easy install, with vqmod support
- You don't have to change you code
- All the twig amazing features


How to install it:
************************************************************
1 - Unzip the extensions folder in your desktop ( or any other place you can remmember );
2 - Drag the content of the upload folder into your ftp; ( nothing will be overwritted );
3 - Like VQMOD, to install go to yourstore.com/twig/install ( replace yourstore.com with your store address);

O que a extensão Twig para Opencart faz?
************************************************************
Essa modificação inclúi a template engine twig no OpenCart, a engine irá processar após a interpretação do template pelo opencart, permitindo você utilizar ambos.

Principais Funcionalidades
************************************************************
- Não precisa configurar
- Fácil de instalar, com suporte ao VQMOD
- Não necessita altearação no código
- Todas as funcionalidade do twig

Como instalar
************************************************************
1 - Descompacte a extensão no seu desktop ( ou outro local que possa localizar );
2 - Arraste ou copie o conteúdo da pasta upload para o seu ftp; ( nenhum arquivo do opencart será substituído);
3 - Assim como o VQMOD, para instalar acesse sualoja.com/twig/install ( substitua sualoja.com pelo seu endereço);
